_Note: To run all C code in vscode, have a C/C++ compiler installed._

Type in `gcc <filename> -o output_program && ./output_program`. This generates the output_program file and runs the code immediately in the terminal.

Example:

```bash
gcc area-calculation-messy.c -o output_program && ./output_program
gcc area-calculation-clean.c -o output_program && ./output_program
```
