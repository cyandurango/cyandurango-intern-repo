# 2.5 Set up VS Code and learn keyboard shortcuts

Research VS Code shortcuts and list at least 10 shortcuts that you find useful.
Categorize them into Navigation, Editing, Debugging, and Terminal Commands.

## **_My Top 10 Keyboard Shortcuts List_**

For Navigation:

- `CMD + F` useful for finding certain variables and keywords in a large codebase
- `OPT + G` line selection
- `CMD + arrow up/down` cursor selection inserted above or below selected line

For Editing:

- `SHIFT + OPT + <-/->` useful for selecting certain words and lines of code carefully.
- `CTRL+A` generates comment blocks for documentation purposes
- `OPT + arrow up/down` move whole line of code up or down the codebase

For Debugging

- `F5` start/continue the program
- `SHIFT+F5` stop the program

For Terminal Commands:

- `CMD + arrow up/down` useful in scrolling through terminal output lines.
- ``CTRL + ` `` show/hide integrated terminal

## **_Which shortcuts do you think will be the most useful in your daily work?_**

To me, the editing and navigation shortcuts would be the most useful, as it would certainly help in viewing and changing the code much easier

## **_How can keyboard shortcuts improve your productivity?_**

Keyboard shortcuts takes less time as it keeps your hands in the keyboard at all times, allowing no time lags from moving your hands toward the mouse or touchpad.
